# Code of Conduct

## Standards

Project discussions should stay respectful, technical, and focused on improving
the software. Harassment, threats, personal attacks, and disclosure of private
information are not acceptable.

## Reporting

Report conduct concerns through the repository owner's private contact channels.
For exploitable security issues, follow [SECURITY.md](SECURITY.md) instead of
opening a public issue.

## Enforcement

Maintainers may remove comments, close issues, or restrict participation when
behavior disrupts the project or puts contributors at risk.
